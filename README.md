# UpCraft
简易的软件更新系统

> [!WARNING]  
> 本项目处于极早期开发阶段，API、文档、示例均可能发生大幅变动。欢迎关注。

## 简介
UpCraft是一个简易的软件更新系统。
它包含一个服务端和一个客户端。

服务端负责管理软件的版本信息，并提供给客户端软件更新的版本信息。
客户端通过与服务端的通信，获取最新的软件版本信息，并进行软件的自动更新。

我们计划为开发者提供两种可选的客户端形式：独立客户端版和头文件集成的方式。

## 使用
### 服务端
服务端使用Python编写，依赖Flask框架。

使用方法详见[服务端文档](./server/README.md)。

### 客户端
客户端使用C++编写，目前实现了头文件集成demo。
