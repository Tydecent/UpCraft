# UpCraft 服务端API手册


## 1. 健康监测
**路由**：`/health`

**请求方法**：`GET`

**返回示例**：
```json
{
    "status": "ok"
}
```

**说明**：用于健康监测，返回状态为`ok`、返回码`200`表示服务正常。

## 2. 请求最新版本信息
**路由**：`/api/v1/get_version`

**请求方法**：`GET`

**返回示例**：
```json
{
    "version": "1.0.0"
}
```

**说明**：用于获取最新版本信息，返回码`200`表示请求成功。

## 3. 下载最新更新包
**路由**：`/api/v1/download_update`

**请求方法**：`GET`

**返回**：更新包文件。

**说明**：用于下载最新版本的更新包，返回码`200`表示请求成功。

## 4. 获取指定版本号的更新包
**路由**：`/api/v1/get_version/<version_number>`
其中：
- `<version_number>`：版本号

**请求方法**：`GET`

**返回**：更新包文件。

**说明**：用于下载指定版本的更新包，返回码`200`表示成功。
